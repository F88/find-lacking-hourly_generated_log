
Usage
=====

    Usage : '$0' [-h] [-p FilenamePattern] [yyyy] [mm] [yyyy] [mm]

Options
-----------

### -h

Show usge and exit

### -p "FilenamePattern"

Pattern string of Path to search

"Filename Pattern" must be ...

 * [path-to-dir/]*yyyy*mm*dd*hh*
 * [path-to-dir/]*yyyy*mm*dd*


Args
-----------

* yyyy
 * year(from,to)
* mm
 * month(from,to)
